# System Programming Project

This project is developed as part of the System Programming course.

## Project Objective

The objective of this project is to gain hands-on experience in system programming concepts,
collaborative software development, containerization, and CI/CD practices using modern tools.

## Technologies
- Ubuntu
- C++
- Docker
- Docker Compose
- Git & GitHub

## Project Structure
The project consists of multiple components including a C++ application
and a PostgreSQL database running in Docker containers.

## Development Workflow
- Feature branches are created from `develop`
- Pull requests are reviewed before merging
- Direct commits to `main` and `develop` are restricted

This project follows a weekly milestone-based development plan.
